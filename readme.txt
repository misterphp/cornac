Cornac
Small and fast CMS
http://github.com/misterphp/cornac

Build the projet from the command line:

 php stub.php

The output files "index.php" and "index-d.php" (debug version) are generated in the "./build" directory.
The "build number" (serialized in "./build/build-number.txt") is incremented by each call to "sub.php".
